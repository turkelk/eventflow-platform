# Project Backlog

This backlog is organized into implementation waves. Each wave file contains
detailed issues with descriptions, files to create, and acceptance criteria.

## How to Use with Claude Code

1. Open this repo in Claude Code
2. Tell Claude: "Implement wave 1" (or any specific wave)
3. Claude will read the wave file and implement each issue sequentially
4. After each issue is implemented, Claude will update its status to `Done`
